Сущности бизнес-домена:
Пользователь, Счет, Операция

Веб-интерфейс:
"Users", "Accounts", "Transactions" - списки пользователей, счетов и переводов, соответственно.
Сделать перевод: Transactions - Make a transaction.
История операций по счету: Accounts - details. Там же историю можно экспортировать в CSV-файл за заданный период. Файл генерируется в рабочей директории каталога.
